# .pi

This directory keeps prompt-oriented compatibility assets related to the OpenSpec workflow.

## Purpose

- `prompts/` contains prompt files that use the slash-colon command style, such as `/opsx:apply`
- `skills/` contains matching skill definitions for that prompt flow

## Why it is still here

Some content overlaps with `.opencode/`, but the files are not byte-for-byte identical and may serve different invocation styles.

For that reason, this directory is documented rather than deleted.
